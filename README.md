# Surya B - Data Analytics Portfolio

A modern, responsive portfolio website showcasing my skills and projects in Data Analytics, Business Intelligence, and Data Science.

## 🚀 Features

- **Modern Design**: Clean, professional interface with dark/light theme support
- **Responsive Layout**: Optimized for all devices and screen sizes
- **Interactive Components**: Smooth animations and hover effects
- **Contact Form**: Built-in contact form with validation
- **Social Integration**: Links to LinkedIn, GitHub, LeetCode, and HackerRank
- **Project Showcase**: Detailed project cards with technology stacks
- **Skills Section**: Comprehensive skills and expertise display

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **UI Components**: Radix UI primitives with custom styling
- **Build Tool**: Vite
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Theme**: Next Themes for dark/light mode

## 📦 Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd surya-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:8080`

## 🚀 Deployment

The portfolio is designed to be easily deployable to various platforms:

- **Netlify**: Connect your GitHub repository for automatic deployments
- **Vercel**: Deploy with zero configuration
- **GitHub Pages**: Use the build command and deploy to GitHub Pages

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory, ready for deployment.

## 🎨 Customization

### Personal Information
Update your personal details in the following components:
- `src/components/Hero.tsx` - Name, title, description, contact info
- `src/components/Contact.tsx` - Contact information and social links
- `src/components/Education.tsx` - Education and certifications
- `src/components/Projects.tsx` - Your projects and work

### Styling
- Colors and themes can be customized in `tailwind.config.ts`
- Component styles are in individual component files
- Global styles are in `src/index.css`

### Content
- Replace `src/assets/profile.jpg` with your profile picture
- Update `public/Surya Resume Updated.pdf` with your resume
- Modify project descriptions and links in `Projects.tsx`

## 📱 Responsive Design

The portfolio is fully responsive and optimized for:
